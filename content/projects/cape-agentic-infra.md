# CAPE Agentic Infrastructure

CAPE Agentic Infrastructure explores how LLM-assisted agents can support infrastructure provisioning and proactive edge-cloud automation. The work is aligned with my current focus on AI workflow orchestration, agents, and infrastructure-aware automation.

## Contribution

- Developed Python prototypes for infrastructure-oriented agent workflows.
- Worked with FastAPI, SQLAlchemy, Pydantic, and LangGraph-style orchestration patterns.
- Contributed to structured representations of infrastructure requests and provisioning decisions.
- Evaluated how LLM-based reasoning can support infrastructure automation while keeping outputs inspectable.

## Technical Context

- **Agent layer**: LLM-assisted workflows and structured reasoning.
- **Backend**: Python, FastAPI, SQLAlchemy, Pydantic.
- **Infrastructure focus**: Edge, cloud, and provider-agnostic provisioning concepts.
- **Research context**: CAPE EU research project.

## Relevance

This project is directly connected to AI orchestration work: using agents and structured automation to bridge human intent, platform APIs, and infrastructure constraints.
